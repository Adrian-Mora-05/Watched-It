# Watched-It

## Requirements

- [Node.js](https://nodejs.org) v18+ (includes npm and npx)
- [Expo Go](https://expo.dev/go) installed on your phone (App Store / Play Store)

## Setup
```bash
cd mobile
npm install       # installs project dependencies
npx expo start    # starts the development server
```

Scan the QR code with your phone camera (iOS) or the Expo Go app (Android).
